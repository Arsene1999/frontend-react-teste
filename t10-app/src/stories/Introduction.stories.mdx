# Sobre a aplicação

Está aplicação tem como objetivo fazer requizições api CatFacts 
e permitir a listagem de seu elementos. 


# A aplicação é composta por:

- React.js
- Typescript
- Chakra.UI
- Api CatFacts
- Axios
- Storybook

# Obs: 
Devido a versatilidade do Chakra.UI não foi preciso utilizar outras libs de css 
como o Styled-Components e SASS(ou SCSS).

# Obs2: 

Infelizmente não estou conseguindo fazer o css do Chakra aparecer no storybook, 
devido ao pouco conhecimento que tenho dessa integração entre os dois. Devido a isso 
tentarei fazer somente uma pequena introdução sobre cada componente e mostrar seus argumentos.
